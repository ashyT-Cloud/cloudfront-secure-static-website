# Secure Static Website Hosting on AWS using Terraform & CloudFront

## 📌 Project Overview

This project demonstrates how to securely host a static website on AWS using modern DevOps practices.

The infrastructure is provisioned with Terraform and follows AWS security best practices by keeping the S3 bucket private and allowing access only through Amazon CloudFront using Origin Access Control (OAC).

## 🔐 Security Features

- Private S3 Bucket
- Block Public Access Enabled
- Bucket Owner Enforced
- Server-Side Encryption (AES256)
- Bucket Versioning
- CloudFront Origin Access Control (OAC)
- Terraform Remote Backend
